A2A Work Group NXM is now public.

It is a multi-agent collaboration skill for OpenClaw that packages:
- project initialization
- role-based routing
- A2A dispatch
- ACK / BLOCKED / timeout / received tracking
- long-task checkpoint management

What makes this version different:
- public-ready cleanup completed
- private team names removed
- machine-specific paths removed
- example configs added
- safer to adopt in third-party environments

GitHub:
<https://github.com/jiangwill2023/a2a-work-group-nxm>

If you are building multi-agent workflows on OpenClaw, this should be a useful starting point.
