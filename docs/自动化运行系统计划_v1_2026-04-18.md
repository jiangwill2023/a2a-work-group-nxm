## 5. project_create.py 目标

输入：
- `project_name`
- `shortname`
- `requester`
- `pm_owner`
- optional supports / assistants / finance support

约束：
- 如果项目原名是中文，脚本落盘前必须转成英文小写 shortname
- 最终 project root / folder name / project_id 必须为英文小写路径

输出：
- 创建 `/Volumes/Local Drawer/SharedProjects/<project_id>/`
- 初始化标准目录与文件
- 预填 `brief.md`
- 预填 `status.json`
- 预填 `next-step.md`
- 预填 `decision-log.md`
- 返回结构化创建结果

---

## 6. 派单自动化目标

- `expected_reply`
